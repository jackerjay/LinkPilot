# LinkPilot Browser Extension

Manifest V3 extension that observes browser-internal navigation and asks the
LinkPilot daemon to re-route mis-targeted links. Ships in **v0.3**; this
directory is reserved.

Target browsers: Chrome, Arc, Microsoft Edge (Chromium MV3), then Firefox
(WebExtensions) in v0.4.

See `docs/linkpilot-design-v0.1.md` §7.2 – §7.4 and §10 for the design.
